⚙️ Falta conectar tu backend

Esta página lee el índice de Drive a través de tu Worker de Cloudflare (que guarda la API key de forma segura).

  1. Despliega el Worker en backend/ siguiendo su README (npx wrangler deploy).
  2. Copia la URL que te da Wrangler, ej: https://drive-index-api.tu-cuenta.workers.dev
  3. Pégala abajo donde dice PEGA_AQUI_TU_WORKER_URL en la constante API_BASE.
  4. Guarda y recarga.
✅ La API key de Google ya no vive en este HTML — queda como secret dentro del Worker.
